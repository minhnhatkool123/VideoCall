# VideoCall
- Tạo phòng nhanh chóng dễ dàng
- Gọi video, nhắn tin trực tiếp
- Có thể tắt mic và cam
- Theo dõi được số lượng người trong phòng
- Số lượng tối đa 50 người 1 phòng
